static

not positioned in anyway; positioned according to normal flow of page

relative

positioned relative to its normal position

absolute

positioned relative to the nearest positioned parent instead of viewport

fixed

positioned relative to the viewport; always in the same place even if the page is scrolled

absolute

if not a child of positioned parent, uses document body and moves along with page scrollings

sticky

positioned based on user's scroll position; is relative until a given offset position is met in the viewport - then "sticks" in place like fixed element
